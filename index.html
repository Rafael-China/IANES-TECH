O que é html e css?

O html são os códigos que serão enviados para o navegador para com que ele possa interpretar e traduzir esse tipo de linguagem, assim o navegador fazendo esse papel de tradutor exibindo assim a página para o usuário.
Ver mais... Primeiro para você poder ver esses códigos pressione com o botão direito na página e clique em “Exibir código fonte” ou “Inspecionar Elemento” assim ficando visível as linhas de código da página em na qual você se encontra.

Qual Ferramenta utilizaremos?

Para nossa didática faremos uso do Visual Studio Code, então portanto pedimos que vocês o tenham instalado para podermos dar seguimento.
Ver mais... Nosso primeiro passo será criar um arquivo então para isso faça uma pasta na qual você irá armazenar os seus arquivos, e no Visual Studio Code no canto superior esquerdo você deve selecionar essa pasta, dentro dela você deve criar um arquivo com chamado “index.html” que vai ser o nosso arquivo principal, com o uso da tecla f5 nós poderemos executar esse arquivo a qualquer momento assim vendo oque há nele, mas nesse momento não veremos nada por não haver conteúdo nele.
